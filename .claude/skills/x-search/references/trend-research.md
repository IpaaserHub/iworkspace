# トレンド調査マニュアル

X上の投稿を活用して、特定のトピックやキーワードのトレンド・世論・盛り上がりを多角的に調査するためのリファレンス。

## 概要

単発のキーワード検索ではなく、トレンドの背景・文脈・温度感・主要プレイヤーまで段階的に掘り下げるリサーチプロセス。

## フロー

### Phase 1: トレンドの特定

ユーザーがトピックを明示している場合はそのまま進む。曖昧な場合は AskUserQuestion で確認する。

```
AskUserQuestion:
  "トレンド調査を行います。以下を教えてください：
   - 調べたいトピック/キーワード
   - 業界やカテゴリ（例: AI、SaaS、暗号資産、ゲームなど）
   - 期間（例: 直近1週間、今月、特定の日付以降）
   - 日本語圏・英語圏・両方のどれを対象にするか"
```

確認すべき情報：
- トピック/キーワード
- 業界・カテゴリの絞り込み
- 期間指定
- 言語圏（日本語/英語/両方）
- 特に知りたい観点（技術動向、世論、ビジネスインパクトなど）

### Phase 2: 広域スキャン（全体像の把握）

トピックが確定したら、**4つの角度**から並行検索して全体像をつかむ。

#### 検索パターン

1. **メインキーワード検索**
   - トピックの直接的なキーワードで検索
   - 直近の投稿量・反応の大きさでトレンドの温度感を把握

2. **インフルエンサー・専門家の発言**
   - そのトピックについて発信力のあるアカウントの投稿
   - 業界リーダー、アナリスト、メディアの見解

3. **関連キーワード・ハッシュタグ検索**
   - メインキーワードに関連するサブトピックや派生ワード
   - 関連するハッシュタグの動向

4. **一般ユーザーの反応・意見**
   - 賛否、感想、体験談などリアルな声
   - 盛り上がりの質（期待/不安/批判/称賛）

```
# 並行実行の例（最大4つ）
Agent tool 1: "X検索: {トピック}メインキーワード直近投稿"
Agent tool 2: "X検索: {トピック}専門家・インフルエンサーの発言"
Agent tool 3: "X検索: {トピック}関連キーワード・ハッシュタグ"
Agent tool 4: "X検索: {トピック}一般ユーザーの反応"
```

### Phase 3: ユーザーへの中間報告 & 深掘り方向の確認

広域スキャンの結果を共有し、次に何を深掘りするかを確認する。

```
AskUserQuestion:
  "{トピック} のトレンドをざっと調べました。概要は以下の通りです：

   📌 トレンドの温度感:
   - {要約: 例）直近1週間で急激に話題が増加、特に3/8以降}

   📌 主要な論点:
   - {要約: 例）技術的な可能性への期待と、倫理面の懸念が二極化}

   📌 キープレイヤーの発言:
   - {要約: 例）〇〇のCEOが支持を表明、△△は慎重な姿勢}

   📌 一般ユーザーの反応:
   - {要約: 例）ポジティブ寄りだが、具体的なユースケースを求める声が多い}

   さらに深掘りしたい方向はどれですか？"

  options:
    - "時系列での変化を追いたい"
    - "賛成派 vs 反対派の論点を詳しく"
    - "具体的なユースケースや事例を知りたい"
    - "その他（具体的に指定）"
```

### Phase 4: ディープダイブ（深掘り検索）

ユーザーが選んだ方向に応じて、具体的な検索を追加実行する。

#### パターン別の深掘り検索

**時系列での変化**
- 期間を区切って（例: 週ごと、月ごと）同じキーワードで検索
- いつ話題が始まったか、ピークはいつか、現在は上昇/下降傾向か
- きっかけとなったイベントや投稿の特定

**賛成派 vs 反対派の論点**
- ポジティブなキーワード（期待、すごい、革新的、便利）で検索
- ネガティブなキーワード（懸念、問題、危険、反対）で検索
- 両陣営の代表的なアカウント・投稿を特定

**具体的なユースケース・事例**
- `"{トピック} 使ってみた / 導入した / 試した"` で検索
- `"{トピック} 事例 / 活用 / 実装"` で検索
- 企業の導入事例や個人の体験談を収集

**その他のユーザー指定方向**
- ユーザーの指定に応じて柔軟に検索クエリを設計

### Phase 5: 最終レポート

すべての検索結果を統合し、以下の構成でレポートをまとめる。

```markdown
# トレンド調査レポート: {トピック}
## 調査日: {日付}

### エグゼクティブサマリー
- 3-5行でトレンドの現状と注目ポイントを要約

### 1. トレンド概況
- 話題の規模感（投稿量、反応数の傾向）
- いつ頃から盛り上がっているか
- 現在のフェーズ（黎明期/成長期/ピーク/安定/下降）

### 2. 主要な論点・テーマ
- トピックに関する主な議論ポイント
- 賛否が分かれている点

### 3. キープレイヤーの動向
- 影響力のあるアカウントの発言まとめ
- 業界リーダー、メディア、アナリストの見解

### 4. 一般ユーザーの声
- ポジティブな反応
- ネガティブな反応・懸念
- 全体的なセンチメント

### 5. 注目すべきポイント・示唆
- 今後の展開予測
- ビジネスや意思決定に活かせるインサイト
```

レポートは `/Users/shotanakayama/projects/iworkspace/workspace/x-results/reports/` に保存する。

## 注意事項

- 各Phaseで検索の並行実行は**最大4つまで**
- Phase 3の中間報告は省略しない（ユーザーの関心に沿った深掘りが目的）
- 検索言語はユーザーの指定に従う。指定がなければ日本語と英語の両方を検討
- トレンドの「温度感」を伝えることを意識する（数値だけでなく質的な評価も）
- バイアスに注意：X上の声は必ずしも世論全体を代表しない旨をレポートに含める
